# StockFlow Dashboard - User Guide

## Quick Overview
**StockFlow Dashboard** is a comprehensive inventory management system designed to help businesses track, manage, and optimize their stock levels in real-time. The platform offers web and mobile access with powerful analytics, automation features, and seamless integrations.

---

## Getting Started

### 1. **Account Creation & First Login**
- **Access Application**: Navigate to StockFlow Dashboard web URL
- **Create Account**: Click "Sign Up" if you don't have an account
  - Enter business email address and create secure password
  - Verify email address via confirmation link
  - Accept terms of service and privacy policy
- **First Login**: Sign in with your new credentials

### 2. **Onboarding Wizard**
Complete the guided setup process:

**Step 1 - Organization Setup**
- Enter company name and business details
- Set time zone and currency preferences
- Upload company logo (optional)

**Step 2 - User Profile**
- Set display name and contact information
- Configure notification preferences (email, mobile push)
- Choose interface theme (light/dark mode)

**Step 3 - Initial Categories**
- Create product categories (e.g., "Electronics", "Office Supplies")
- Set default minimum stock thresholds
- Configure units of measure for different product types

**Step 4 - Team Setup** (optional)
- Invite team members with appropriate roles
- Set up user permissions and access levels

### 3. **Loading Your Initial Data**

#### **Option A: Excel Import**
1. **Download Template**: Click "Import" → "Download Excel Template"
2. **Prepare Data**: Fill template with your inventory data:
   - Item Name, SKU, Barcode, Category
   - Current Quantity, Cost Price, Selling Price
   - Minimum Threshold, Location, Description
3. **Upload File**: Click "Import from Excel" → Select your file
4. **Review & Confirm**: Check import preview → Click "Import Items"
5. **Validation**: System reports success/errors and item count

#### **Option B: Google Sheets Integration**
1. **Connect Google**: Go to Settings → Integrations → "Connect Google Sheets"
2. **Authorize Access**: Login to Google account and grant permissions
3. **Select Sheet**: Choose your inventory spreadsheet from Google Drive
4. **Map Columns**: Match spreadsheet columns to inventory fields
5. **Import Settings**: Choose one-time import or ongoing sync
6. **Execute Import**: Start data import process

#### **Option C: Zoho Integration**
1. **Access Integrations**: Navigate to Settings → Integrations
2. **Connect Zoho**: Click "Connect to Zoho Books/Inventory"
3. **Authentication**: Login to Zoho account and authorize StockFlow
4. **Map Products**: Match existing Zoho items to StockFlow categories
5. **Sync Settings**: Configure bi-directional sync preferences
6. **Initial Sync**: Start first synchronization of inventory data

#### **Option D: Manual Entry**
- **Add Items One-by-One**: Click "Add New Item" button
- **Bulk Add**: Use quick-add form for multiple similar items
- **Barcode Scanning**: Use camera to scan existing product barcodes

### 4. **Mobile Device Setup (Android)**

#### **APK Installation**
1. **Download APK**: Get installation file from your administrator
2. **Enable Installation**: Android Settings → Security → "Unknown Sources" → Enable
3. **Install App**: Tap downloaded APK file → Follow installation prompts
4. **Open App**: Find "StockFlow Dashboard" in app drawer

#### **Mobile Configuration**
1. **Login**: Use same credentials as web application
2. **Grant Permissions**: 
   - **Camera Access**: Required for barcode scanning
   - **Storage Access**: For offline data and photo uploads
   - **Network Access**: For data synchronization
3. **Notification Setup**: Enable push notifications for alerts
4. **Offline Settings**: Configure data sync preferences
5. **Test Scanner**: Try scanning a barcode to verify camera works

### 5. **Dashboard Overview**
The main dashboard provides:
- **Real-time stock levels** and key metrics
- **Low stock alerts** for items requiring attention
- **Recent activity** logs and updates
- **Analytics cards** showing business insights
- **Priority items** highlighting critical stock

## Initial Setup Troubleshooting

### **Account Creation Issues**
- **Email not received**: Check spam folder, use business email address
- **Password requirements**: Must be 8+ characters with numbers and symbols
- **Organization name conflicts**: Choose unique organization identifier

### **Data Import Problems**
- **Excel import fails**: Use provided template exactly, check file size (max 50MB)
- **Google Sheets connection**: Ensure proper Google account permissions
- **Zoho integration issues**: Verify Zoho account has API access enabled
- **Duplicate items**: System will flag duplicates during import preview

### **Mobile App Setup Issues**
- **APK won't install**: Enable "Unknown Sources" in Android security settings
- **Camera not working**: Grant camera permissions in app settings
- **Sync problems**: Check internet connection and login credentials
- **App crashes**: Ensure Android 8.0+ and sufficient storage space

### **Integration Connection Problems**
- **Zoho authentication fails**: Re-authorize access in Zoho account settings
- **Google Sheets not syncing**: Check sharing permissions on spreadsheet
- **Data mapping errors**: Review column mapping during integration setup
